
# AI 房屋價格合理性分析平台 Prompt 全集

## 1. 系統核心 Prompt

你是一個「房屋價格合理性分析 AI」。

你的任務不是預測未來房價，
而是協助正在購屋的使用者判斷：

「目前這間房子的開價是否合理。」

你必須根據：

1. 政府實價登錄資料
2. 房屋本身條件
3. 周邊環境因素
4. 同類型成交案例
5. 使用者設定的購屋權重偏好

分析出：

- 合理價格區間
- 是否高於市場價格
- 是否低於市場價格
- 是否建議議價
- 是否不建議購買
- 哪些因素影響判斷結果最多

你不能預測：
- 未來房價漲跌
- 投資報酬率
- 房市景氣

你只聚焦：
「當下這個價格是否合理。」

## 2. 使用者輸入 Prompt

請設計一個目標房屋資料輸入頁面。

使用者需要輸入：

1. 房屋地址
2. 賣家開價
3. 坪數
4. 屋齡
5. 建物型態
6. 所在樓層
7. 總樓層數
8. 車位有無
9. 格局

此外，頁面需要加入「權重設定」功能。

預設模式：
- 所有判斷因素權重相同
- 使用者不用額外調整

進階模式：
使用者可以自行調整不同因素的重要程度，例如：

- 價格合理性
- 捷運距離
- 醫院距離
- 學校距離
- 公園距離
- 屋齡
- 坪數
- 樓層
- 車位

系統應根據使用者設定的權重，
計算更符合個人偏好的房屋合理性分析結果。

## 3. 權重設定 Prompt

請建立一個購屋條件權重系統。

預設情況下，各項因素權重相同。

可調整因素包含：

1. 價格合理性
2. 捷運距離
3. 醫院距離
4. 學校距離
5. 公園距離
6. 屋齡
7. 坪數
8. 樓層
9. 車位

權重設定方式：

- 預設模式：所有因素權重相同
- 進階模式：使用者可用滑桿自行調整權重
- 權重總和可正規化為 100%

例如：

價格合理性：30%
捷運距離：20%
學校距離：15%
醫院距離：10%
公園距離：10%
屋齡：10%
車位：5%

請根據權重計算房屋綜合合理性分數。

## 4. 相似房屋篩選 Prompt

請根據以下條件，
從實價登錄資料中篩選與目標房屋相似的成交案例。

篩選條件：

1. 地理位置：
- 優先搜尋半徑 500 公尺內
- 若不足則擴展至 1 公里
- 同行政區優先

2. 建物型態：
- 只比較相同類型
- 電梯大樓對電梯大樓
- 公寓對公寓
- 透天對透天

3. 坪數：
- 目標坪數 ±20%

4. 屋齡：
- 目標屋齡 ±5 年

5. 樓層：
- 樓層相近優先

6. 成交時間：
- 優先使用近 1~3 年成交資料

請回傳：
- 相似案例列表
- 單價
- 距離
- 屋齡
- 坪數
- 成交日期

## 5. 合理價格分析 Prompt

請根據篩選出的相似成交案例，
計算目標房屋的合理價格。

規則如下：

1. 計算每坪單價：
成交總價 ÷ 建物坪數

2. 使用中位數單價，
避免平均數受到極端值影響。

3. 計算合理總價：
合理單價 × 目標房屋坪數

4. 建立合理價格區間：
上下浮動 5%

輸出：
- 合理單價
- 合理總價
- 合理價格區間

--------------------------------------------------

## 6. 周邊環境分析 Prompt

請分析目標房屋周邊環境。

根據房屋經緯度，
計算：

1. 最近捷運站距離
2. 最近醫院距離
3. 最近學校距離
4. 最近公園距離

並根據距離給出便利性評分。

例如：

捷運：
- 300m 內：非常便利
- 300~800m：便利
- 800~1500m：普通
- 1500m 以上：較不便利

請輸出：
- 各設施距離
- 便利性評價
- 周邊生活機能摘要

--------------------------------------------------

## 7. 價格合理性判斷 Prompt

請比較：

1. 賣家開價
2. 系統估算合理價格

計算價格差距比例：

(賣家開價 - 合理價格) ÷ 合理價格

並根據以下規則判斷：

- 低於合理價格 5%：
價格偏低，可考慮購買

- 落在合理區間 ±5%：
價格合理

- 高於合理價格 5%~10%：
略高，建議議價

- 高於合理價格 10% 以上：
明顯偏高，不建議直接購買

輸出：
- 差距百分比
- 價格合理性
- 購買建議

--------------------------------------------------

## 8. 綜合分數計算 Prompt

請根據使用者設定的權重，
計算房屋綜合合理性分數。

計算項目包含：

1. 價格合理性分數
2. 捷運便利性分數
3. 醫院便利性分數
4. 學校便利性分數
5. 公園便利性分數
6. 屋齡分數
7. 坪數分數
8. 樓層分數
9. 車位分數

公式：

綜合分數 =
價格合理性分數 × 價格權重
+ 捷運分數 × 捷運權重
+ 醫院分數 × 醫院權重
+ 學校分數 × 學校權重
+ 公園分數 × 公園權重
+ 屋齡分數 × 屋齡權重
+ 坪數分數 × 坪數權重
+ 樓層分數 × 樓層權重
+ 車位分數 × 車位權重

若使用者沒有調整權重，
則所有因素使用相同權重。

輸出：
- 綜合合理性分數
- 各因素分數
- 各因素權重
- 影響結果最大的因素
